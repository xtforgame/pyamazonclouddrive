# How to create reports #
```
$ easy_install coverage
$ coverage.py erase
$ coverage.py run test.py
$ date >> coverage.txt
$ coverage report >> coverage.txt
$ coverage html
$ move htmlcov htmlcov_`date '+%Y%m%d'`
```
